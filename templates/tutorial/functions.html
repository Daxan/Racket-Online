Prozeduren und Funktionen

Eine Prozedur oder Funktion wird mittels define benannt. Hinter define folgt, der Name.
Syntax:
(define Name Ausdruck)
(define a 8)

Durch Lambda-Ausdrücke werden Funktionen definiert. Sie erhalten Funktionsargumente und den Funktionsrumpf.
Syntax:
(define Funktionsname (lambda (Arg1 Arg2 ...) (Funktionsrumpf) ))

Dieses Beispiel addiert zu einer Zahl 23:
(define add23 (lambda (x) (+ x 23)))
(add23 5) => 28

Dies lässt sich auch mittels define verkürzt so schreiben:
Syntax:
(define (Funktionsname Arg) (Funktionsrumpf))
(define (add23 x) (+ x 23))
