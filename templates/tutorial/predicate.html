Wahrheitswerte

In Scheme gibt es die Wahrheitswerte true und false, die mit #t bzw. #f symbolisiert werden.

  • Die Auswertung von #f liefert #f.
  • Die Auswertung von #t liefert #t.

die Bedingungen enthaltenden Scheme-Formen als true bzw. false interpretieren. In bedingten Ausdrücken werden nur #f und die leere Liste ()* als false interpretiert, alle anderen Objekte der Typen pair, symbol, number, string, vector, procedure und #t ergeben true:
(if #t 'true 'false) => true
(if #f 'true 'false) => false
(if '() 'true 'false) => true
(if 1 'true 'false) => true
(if '(a b c) 'true 'false) => true
Ein Prädikat ist eine Funktion, die einen Wahrheitswert (true,false) liefert:
(null? '()) => #t
(null? '(a b)) => #f
(or #t #f #t) => #t
(or #f '() #f) => ()
(and #t #f #t) => #f
(and #t '() #t) => #t
